WASD for movement
R: restart
Z: undo (you can undo restarts and level transitions)
Q: interact: When standing on flowers, press Q to enter the level. In the overworld, if you've beaten a level, a different mesh appears - you can still enter the level with Q.
ESC: leave out of a level you've had enough of, without solving it.
TAB: go to alternate, overhead camera (only works in overworld right now, no other levels need an alternative camera)

debug binds:
F: if you want to skip a level, press F while standing on a level instead of Q to 'solve' the level
9: toggles cheating (player and objects have no gravity)
1: toggle debug mode (able to move around with WASD, place/break blocks with left/right click. 
J: toggle to a more friendly FOV for flying around
0: to go back to game mode
backspace: reset camera to start position
